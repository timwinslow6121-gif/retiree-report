# Retiree Report Project Guide

## 🎯 Project Vision
A high-legibility, high-accuracy Medicare & retirement blog. 
**Target Audience:** Seniors (65+). 
**Mandatory Style:** High contrast, large fonts (min 18px), zero "dark patterns."

## 🛠 Tech Stack
- **Structure:** Pure HTML5 (15 existing pages).
- **Styling:** `main.css` (global) and `pages.css` (specific overrides).
- **Knowledge:** Obsidian Vault (via MCP) containing `2026 Medicare Rules`.

## 📜 Workflow Rules (IMPORTANT)
1. **Accuracy First:** Always cross-reference Medicare numbers against Obsidian before saving.
2. **GSD Framework:** Use `/plan` to define changes, then `/execute` to apply them.
3. **Design System:** Use UI UIX Pro Max to maintain accessibility standards.
4. **No Hallucinations:** If 2026 data isn't in the vault, ASK the user to provide it.

## 📂 Navigation
- `/`: Homepage and index
- `/css/`: All styling
- `[Obsidian Vault]`: Source of truth for Medicare laws and Brand Skill.
