<task type="auto">
  <name>Task 1: Add mobile-always-visible Subscribe CTA to base.njk header bar</name>
  <files>src/_includes/layouts/base.njk, assets/css/main.css</files>
  <action>
In `src/_includes/layouts/base.njk`, insert a new Subscribe Free anchor element between the `<button class="nav-toggle">` element and the `<div class="nav-menu">` div.

The insertion should look like this inside `.site-header__inner`:
```html
<a href="#newsletter" class="btn-primary nav-header-cta" aria-label="Subscribe to the free newsletter">Subscribe Free</a>
```

Place it AFTER the closing `</button>` of `.nav-toggle` and BEFORE `<div class="nav-menu"...>`.

Use `class="btn-primary nav-header-cta"` — the locked v3.0 `.btn-primary` class provides the base button styles; `nav-header-cta` is a positioning/sizing modifier that adds compact dimensions appropriate for the header bar.

Also add this CSS block to `assets/css/main.css` after the existing .nav-toggle rules:
```css
/* Mobile header Subscribe CTA — always visible */
.nav-header-cta {
  display: inline-flex;
  align-items: center;
  padding: 8px 14px;
  font-family: var(--font-sans);
  font-size: var(--text-xs);
  font-weight: 700;
  letter-spacing: 0.1em;
  text-transform: uppercase;
  text-decoration: none;
  border-radius: var(--radius-sm);
  background: var(--color-gold);
  color: var(--color-navy);
  white-space: nowrap;
  flex-shrink: 0;
}

.nav-header-cta:hover {
  background: var(--color-gold-dark);
  color: var(--color-navy);
}

/* On desktop (768px+), hide the duplicate — the in-menu CTA is sufficient */
@media (min-width: 768px) {
  .nav-header-cta {
    display: none;
  }
}
```

Note: On desktop the `.nav-menu-cta` inside nav.njk (inside the horizontal nav row) serves as the CTA. The `.nav-header-cta` is mobile-only. On mobile, the in-menu `.nav-menu-cta` remains for when the menu is open — the header-bar CTA is what makes it visible when the menu is closed.
